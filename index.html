使用Atom进行Markdown写作

发表于 2017-10-07 |

这里记录用到的几个Markdown相关的Atom插件

1. markdown-image-paste

  1. 使用微信等工具截图保存在粘贴板中
  2. 在空行中输入希望图片保存的文件名或直接按ctrl+v
  3. 图片将自动插入在当前位置并与.md文件保存在一起
  4. gif只能手动插入

2. markdown-preview-plus

用于替代Atom自带的markdown插件markdown-preview.
ctrl+shift+m进行预览实时渲染

3. markdown-scroll-sync

预览视图随编辑视图实时刷新,与光标所在位置同步.
解决编辑/预览不同步的烦恼

4. markdown-table-editor

快速生成表格, 使用tab/enter键在列/行间切换

Atom使用配置

Atom设置中，Editor-Invisible-Show Invisibles选项会将不可见符号用特殊字符显示出来。
如将空格显示为“.”，tab显示为“⇥”，同时回车符也会显示出来，显得页面很杂乱。设置里并不能单独选择
是否打开某个字符的显示。解决办法：Atom-Config 添加如下内容
```
“*”:
editor:
invisibles:
cr: “”
eol: “”
space: “.”
tab: “⇥”
showInvisibles: true
showIndentGuide: true #显示缩进辅助线
